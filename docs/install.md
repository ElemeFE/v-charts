### 安装

#### npm安装

```
npm i v-charts echarts -S
```

#### cdn

```html
<script src="//unpkg.com/vue/dist/vue.js"></script>
<script src="//unpkg.com/echarts/dist/echarts.min.js"></script>
<script src="//unpkg.com/v-charts/lib/index.min.js"></script>
```
